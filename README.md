##.env file

This project need a .env file to run. Just create a file call .env in the src folder. This file should look like this:

```.env
NODE_ENV=development
DB_HOST=localhost
DB_PORT=3306
DB_USERNAME=root
DB_PASSWORD=password
DB_NAME=db_name

```
